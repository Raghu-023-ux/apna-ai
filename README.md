# 🛡️ Cognitive OS: The Ocular Sentinel Protocol
*(Merge of Active Eye Defense & Cognitive OS)*

An AI-driven, clinical-grade desktop application designed to combat Digital Eye Strain (Computer Vision Syndrome). This project merges the cybernetic telemetry monitoring of **Active Eye Defense** with the clinical interactive therapy modules of **Cognitive OS**, creating the ultimate hackathon solution for ocular health.

## 🚀 Key Innovation: The Guardian Protocol
Standard blink reminders are passive. **Cognitive OS Sentinel** is proactive. It monitors your neural telemetry in real-time through your webcam and enforces a 3-stage clinical therapy protocol when severe visual strain is detected.

### 1. The Integrative Fatigue Index (IFI)
Unlike simple blink counters, our React-based AI model calculates visual strain using multiple telemetry variables:
- **Blink Density Decay:** Real-time monitoring of your blinks per minute (BPM).
- **Fixation Jitter:** Measuring "micro-saccades" or eye jitter to determine when your eyes are struggling to maintain focus.
- **Continuous Stare Duration:** Penalizing prolonged periods of screen locking without relaxation.

### 2. The 3-Stage Clinical Therapy Combo
When the IFI hits critical levels, the "Sentinel" triggers a mandatory, gamified recovery cycle derived from established optometry practices:
- **Phase I: Saccadic Taps:** Rapidly stretches the six extraocular muscles through wide-angle, mathematically validated gaze jumps across all four quadrants of the screen.
- **Phase II: Palming Session:** Forces the user to completely block light from their eyes, monitored via face occlusion, allowing the photoreceptors to fully reset. 
- **Phase III: Focus Shifting (20-20-20 Rule):** Utilizing facial rotation matrices, the system ensures the user looks away from the screen to completely break "accommodation lock" in the ciliary muscles.

## 🛠️ Technical Stack
- **Core Framework:** React 18, Vite
- **AI Engine:** MediaPipe Face Landmarker V2 (Edge AI, running purely client-side)
- **Math Logic:** Euclidean distance-based Eye Aspect Ratio (EAR) calculation, Pitch/Yaw facial rotation matrices.
- **UI/UX:** Cybernetic glass-morphism, dark-mode radial gradients, CSS animations, and real-time biometric telemetry streaming.
