# 🛡️ Active Eye Defense: The Ocular Trifecta Protocol

An AI-driven, clinical-grade eye recovery application designed to combat Digital Eye Strain (Computer Vision Syndrome) using real-time Computer Vision and Predictive Fatigue Modeling.

## 🚀 Key Innovation: The "Ocular Trifecta"
Standard blink reminders are passive. **Active Eye Defense** is proactive. It monitors your neural telemetry and enforces a 3-stage recovery protocol when severe strain is detected.

### 1. The Integrative Fatigue Index (IFI)
Unlike simple blink counters, our AI model calculates strain using multiple variables:
- **Blink Density Decay:** Monitoring the reduction in blink frequency over time.
- **Fixation Stability:** Measuring "micro-saccades" or jitter during prolonged staring.
- **Stare Duration:** Detecting the duration of continuous ocular focus.

### 2. The 3-Stage Therapy Combo
When the IFI hits 75%, the "Sentinel" triggers a mandatory recovery cycle:
- **Phase I: Nebula Saccades (Warmup):** Rapidly stretches the six extraocular muscles through wide-angle gaze jumps.
- **Phase II: Celestial Infinity (Smooth Pursuit):** Relaxes the focal muscles by forcing the eyes to track a 3D infinity loop while enforcing head stillness.
- **Phase III: Depth Recalibration (Accommodation):** Exercises the ciliary muscles by shifting focus across virtual depths, breaking "accommodation lock."

## 🛠️ Technical Stack
- **Core Engine:** MediaPipe Face Landmarker V2 (Edge AI)
- **Math Logic:** Euclidean distance-based Eye Aspect Ratio (EAR) calculation.
- **UI/UX:** Modern Glass-morphism, dark-mode radial gradients, and real-time telemetry streaming.
